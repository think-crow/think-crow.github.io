# 本站架构
本站目前采用 [Hugo](https://gohugo.io/) 作为后台系统，网站主题是 [hugo-ivy](https://github.com/yihui/hugo-ivy) 主题。这个主题是开源的，由[谢益辉](https://yihui.org)所编写，任何人都可以自由使用，所有页面都基于 Markdown 创建，源文件可以在 GitHub 上找到；
